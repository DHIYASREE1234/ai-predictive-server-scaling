# AI Predictive Server Scaling

An AI-powered predictive server scaling system that forecasts workload and automatically recommends server scaling decisions using Machine Learning and Spring Boot.

---

## 📌 Project Overview

Modern cloud applications experience fluctuating workloads. Traditional auto-scaling reacts **after** CPU or memory usage increases.

This project predicts future workload using Machine Learning so that infrastructure can scale **before** performance degrades.

---

## 🎯 Objectives

- Predict incoming server workload
- Recommend proactive scaling decisions
- Reduce infrastructure cost
- Improve application availability
- Build an industry-level AI + Backend project
